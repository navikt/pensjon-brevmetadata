== Innhold

Dette er dokumentasjonen til Pensjon Brevdata som er generert ved bruk av Asciidoc og PlantUML.

Pensjon-brevdata er en applikasjon som inneholder xsd'er og informasjon om brev som benyttes av Pesys i bestilling av brev.
Applikasjonen inneholder tjenester for å hente ut denne informasjonen.

Det er videre lagt til ADR for å støtte om beslutninger som er under vurdering eller forslag til applikasjonen.
